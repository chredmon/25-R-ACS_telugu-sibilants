# 25-R-ACS_telugu-sibilants

This study demonstrates that the ternary sibilant system described in the literature on Telugu is no longer present, and has shifted to a binary alveolar-retroflex system, with the palatal shifting in different directions depending on speaker and item.



## File structure

All project data is in the `dat` folder, with the extracted parameters in `dat/working` and the raw files in `dat/raw`.

All source code for generating analyses and figures is located in `src`.
